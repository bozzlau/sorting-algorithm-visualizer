# Sorting Algorithm Visualizer

一个用于学习排序算法原理的静态网页演示程序。页面通过柱状图动画展示冒泡排序、插入排序、选择排序的执行过程，并在左侧同步高亮对应的 C++ 核心代码。

## 功能特点

- 支持冒泡排序、插入排序、选择排序三种算法。
- 支持自定义排序数据，使用英文逗号分隔，例如 `8, 3, 5, 1, 9, 4`。
- 支持随机生成测试数据。
- 支持开始、暂停、下一步、重置操作。
- 按下右方向键 `→` 可以执行下一步。
- 动画与 C++ 核心代码同步高亮，便于对照理解算法逻辑。
- 插入排序使用“悬浮 key + 空位移动 + 落入插入位”的动画方式展示。
- 项目为纯前端静态页面，不依赖后端或构建工具。

## 运行方式

直接用浏览器打开项目中的 `index.html` 即可运行。

也可以在项目目录启动本地静态服务：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/index.html
```

## 使用说明

1. 在左侧选择排序算法。
2. 在“排序数据”中输入数字列表，点击“应用数据”。
3. 点击“开始”自动播放，或点击“下一步”逐步查看。
4. 观察右侧柱状图动画，同时对照左侧 C++ 代码高亮。

## 支持的数据格式

- 使用英文逗号分隔数字。
- 至少输入 2 个数字，最多 30 个数字。
- 数字范围建议控制在 `-999` 到 `999`，以保证柱状图显示清晰。

示例：

```text
5, 4, 3, 2, 1
1, 2, 3, 4, 5
3, 8, 5, 1, 9, 4
```
